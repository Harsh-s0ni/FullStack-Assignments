# Hospital Website Landing Page

Welcome to the Hospital Website Landing Page project! This showcase demonstrates my web development skills and a commitment to modern design principles. The website has been carefully crafted using contemporary techniques, including flex grid layouts, to provide an engaging user experience.

## Features

- **Modern Design:** The website follows the latest design trends seen in modern healthcare websites, ensuring a visually appealing and professional look.

- **User-Friendly Interface:** The user interface prioritizes ease of navigation, allowing visitors to quickly find essential information about the hospital's services, doctors, and facilities.

- **Clear Information Hierarchy:** The content is organized hierarchically, enhancing readability and ensuring that users can quickly locate the information they are seeking.

- **Engaging Visuals:** High-quality images and graphics have been strategically placed to create a positive and welcoming atmosphere for visitors.

## Technologies Used

- **HTML5:** The foundation of the website's structure and content.

- **CSS3:** Styling and layout enhancements, including flexboxes and grids.

## Installation

1. Clone the repository
2. Open the `Project-2.html` file in your preferred web browser.

## Usage

Feel free to explore the landing page to get a sense of the web development skills and design sensibilities applied in this project. Click on different sections to learn about the hospital's services, staff, and facilities.
